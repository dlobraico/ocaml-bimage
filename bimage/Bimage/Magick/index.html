Module Bimage.Magick

Magick defines image I/O operations using ImageMagick/GraphicsMagick on the command-line

val read : string ‑> ('a, 'b) Bigarray.kind ‑> 'c color ‑> ('a, 'b, 'c) Image.t option

read filename kind color loads an image from filename on disk using the given kind and color

val write : string ‑> ('a, 'b, 'c) Image.t ‑> unit

write filename image saves an image to filename

val command : string Pervasives.ref

command contains the command used to call out to ImageMagick/GraphicsMagick. For example, if you'd like to use GraphicsMagick then set this to "gm convert"
